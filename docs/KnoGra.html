Knowledge Graph(KnoGra)

ナレッジをグラフ図の形式で残す、方向性の策定とかツールの案

  • 残し方

  • 活用の仕方 Knoredge Graphの元になるデータの雛形(knodat)を定義して、 それに基づいたKnoredge Graph(knogra)の学習と形式、利用のためのツールを作りたい。

想定構造

knogra/  
|- __init__.py
|- knodat.py  # loaderも兼ねる？
| |- class data()  
|- knogra.py  
| |- class graph()
|- utils.py
